# Shaft (Pixiv 第三方客户端)

## 1.功能
* 用户登录，新用户注册
* 大陆用户支持直连（详见https://github.com/Notsfsssf/Pix-EzViewer）
* 非会员用户支持按热度排序，按收藏量筛选
* GIF支持播放、保存
* 本地保存浏览历史、下载历史
* 支持多用户切换
* 新增小说功能
* 插画，漫画，小说排行榜，支持按日查看
* 正常收藏，关注
* R80显示（需自行到官网开启）
